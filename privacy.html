Jupnich Privacy Policy

Effective date: April 25, 2026

Overview

Jupnich is a Discord server operations and structure bot. This policy explains what data Jupnich may store or process to provide its features.

Data Jupnich May Store

Depending on how a server uses Jupnich, the bot may store Discord server IDs, channel IDs, role IDs, user IDs, configuration settings, ticket records, moderation records, event records, role panel settings, welcome settings, source routing settings, leveling data, birthday data, and related operational logs.

Message Content

Jupnich may process message content only when needed for enabled features, such as commands, moderation tools, leveling, media tools, tickets, or server operations features. Jupnich does not sell message content.

How Data Is Used

Stored data is used to provide bot features, remember server configuration, support moderation continuity, run ticket/event/role systems, and maintain server operations features.

Data Sharing

Jupnich does not sell user data. Data may be processed by hosting or infrastructure services needed to operate the bot.

Data Removal

Server owners may request removal of stored server data through the official support channel or public project contact method provided with the app.

Security

Reasonable care is taken to protect stored data, but no online service can guarantee perfect security.

Changes

This privacy policy may be updated as Jupnich develops. Continued use of Jupnich after updates means you accept the updated policy.

Contact

For privacy questions or data removal requests, contact the Jupnich developer through the official support channel or public project contact method provided with the app.

Disclaimer

Jupnich is not affiliated with, endorsed by, or operated by Discord Inc.
